# Support The Project

## Report issues

By far the easiest way to help make Gallery better it to use it and report issues and bugs. Found a bug? [Open an issue on GitHub][github-issue].

## Translations

Support the project by localizing on [Weblate](https://hosted.weblate.org/projects/immich/immich/). For more information, see the [Translations](/developer/translations) section.

## Development

If you are a programmer or developer, take a look at Gallery's [technology stack](/developer/architecture.mdx) and consider fixing bugs or building new features. For information about how to contribute as a developer, see the [Developer](/developer/architecture.mdx) section.

[github-issue]: https://github.com/open-noodle/gallery/issues/new/choose
