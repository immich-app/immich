Discovery Page — Layout Options

Four approaches to fitting discovery filters alongside Gallery's existing navigation

A Collapsed Nav Rail + Filter Sidebar
Nav collapses to a 48px icon rail. A 260px filter sidebar sits next to it with temporal picker, people, location, camera, tags, rating, media type. Photos get the remaining space.
Filters always visible — true faceted browsing Temporal picker + filters in natural vertical flow Photos still get ~1100px on 1440p screen
Nav labels hidden — less discoverable for new users ~308px total sidebar width (rail + filters)

B Full Nav + Slide-out Filter Drawer
Nav sidebar stays at full width (160px). Clicking "Filters" opens a 300px drawer that slides over the photo grid from the left. The drawer has a close button and darkens the grid behind it.
Nav always fully visible — consistent with all other pages Photos get maximum width when filters are closed Familiar drawer pattern (Google Photos uses this)
Filters not persistently visible — must open/close drawer Can't see filter changes reflected in real-time in the grid Extra click to access filters each time

C Full Nav + Horizontal Filter Panel
Nav stays full-width. Filters live in a collapsible horizontal strip below the search bar. Temporal picker as a row of year/month chips. Other filters as pill toggles. Expands vertically when sections are opened.
Photos get full width below the filter strip Nav untouched — consistent with all other pages Filter strip can collapse to just active chips
Horizontal space limits filter detail (no room for long people lists) Temporal picker cramped in horizontal layout Eats vertical space — fewer photo rows visible

D Sidebar Switches Between Nav and Filters
The left sidebar has a toggle at the top: "Nav" and "Filters". On the Discover page, it defaults to Filters mode. The sidebar shows filter sections instead of nav items. A small toggle lets you switch back to nav when needed.
Same 160px sidebar width — no extra space used Filters visible by default on Discover page No layout shift compared to other pages
160px is tight for filter content (year chips, people names) Nav hidden when filters showing — extra clicks to navigate away Unconventional pattern — may confuse users
